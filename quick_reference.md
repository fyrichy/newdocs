# Qwen人物融合提示词快速参考

本文档是`qwen_image_edit_character_fusion_guide.md`的精简版，提供关键提示词模板供快速查阅。

---

## 核心原则

1. **明确性原则**：描述越详细，结果越准确
2. **一致性原则**：保持人物特征在多帧/多图中的一致性
3. **上下文原则**：人物必须与场景环境相匹配
4. **细节原则**：描述光线、阴影、视角等技术细节

---

## 任务类型

| 类型 | 说明 | 是否需要掩码 |
|------|------|------------|
| 人物替换 | 将目标图像中的人物替换为指定人物 | 是 |
| 人物添加 | 向图像中添加新的人物 | 是/否 |
| 人物融合 | 将多人特征融合为一人 | 否 |
| 多人物场景融合 | 多人物在同一场景中互动 | 否 |

## 提示词模板

### 单人添加模板

```
【任务类型】人物添加
【添加位置】画面右后方，靠近窗户的位置
【人物设定】
- 身份：28岁女性画家
- 外貌：棕色及肩短发，戴贝雷帽，穿着米色风衣
- 动作：手持画笔和调色板，正在专注地绘画
- 姿态：侧身站立，身体微微前倾
【环境融合】
- 光影：室内自然光，从窗户照入，在人物身上形成柔和光斑
- 阴影：人物在地板上投下自然阴影
- 透视：符合画面透视关系，近大远小
【风格要求】与原图一致的文艺清新风格，色彩温暖
【输出要求】高质量，细节丰富
```

### 多人物场景融合模板

```
【任务类型】多人物场景融合
【场景设定】
- 场景：1920年代复古咖啡馆
- 环境：木质地板，复古吊灯，皮质沙发，墙上挂着老照片
- 光线：暖黄色灯光，温馨氛围
- 色调：复古棕褐色调
【人物1 - 咖啡师】
- 位置：吧台后方
- 身份：40岁男性
- 外貌：八字胡，穿着白色衬衫和黑色围裙
- 动作：正在倒咖啡，面带微笑
【人物2 - 顾客A】
- 位置：左侧靠窗座位
- 身份：25岁女性
- 外貌：波浪卷发，戴珍珠项链，穿红色连衣裙
- 动作：正在阅读一本书，表情专注
【一致性要求】
- 所有人物保持相同的复古风格
- 服装、发型符合1920年代特征
- 光影统一，符合场景灯光设定
【输出要求】8K超详细，电影级照明，复古电影质感
```
