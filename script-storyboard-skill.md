# 剧本分镜生成器

将资料背景转化为剧本故事，并生成分镜提示词和旁白。

## 工作流程

### 步骤 1：剧本创作

当用户提供指令 `以《xxx》为资料背景，进行剧本创作，或者纪录片故事内容创作` 时：

1. 仔细阅读并理解用户提供的资料背景
2. 提炼核心主题、人物、情节线
3. 创作一个完整的剧本/故事，包含：
   - 故事背景设定
   - 主要人物介绍
   - 故事主线（起承转合）
   - 关键场景和冲突

**输出格式：**
```
# 剧本标题

## 背景设定
...

## 人物介绍
...

## 故事主线
### 第一幕：开场
...

### 第二幕：发展
...

### 第三幕：高潮
...

### 第四幕：结局
...
```

---

### 步骤 2：分镜划分

将剧本划分为若干分镜（通常 8-16 个分镜），每个分镜：

1. **分镜编号**：Scene 01, Scene 02...
2. **场景描述**：时间、地点、环境
3. **主要动作**：该分镜的核心事件
4. **人物状态**：角色的表情、动作、心理

---

### 步骤 3：生成中英文分镜提示词

为每个分镜生成 **文生图提示词**，要求：

- **中英文对照**（不是混合），每行先中文再英文
- 保持画面风格一致（统一的光影、色调）
- 确保分镜间视觉连续性
- 下一个分镜要衔接好上一个分镜，不能跑偏、脱节

**格式：**
```
## 分镜 01
【中文提示词】
A wide shot of a misty mountain village at dawn, ancient Chinese architecture, soft golden sunlight streaming through the clouds, peaceful atmosphere, cinematic lighting, 4K, photorealistic

【英文提示词】
广角拍摄黎明时分雾气缭绕的山村，古老中式建筑，柔和的金色阳光穿透云层，和平氛围，电影光效，4K，写实摄影风格
```

---

### 步骤 4：旁白编写

每个分镜后编写旁白，要求：

- 类似讲故事的语气，叙事性强
- 概括当前分镜的内容
- 与前后分镜保持连贯
- 体现故事的节奏和情感

**格式：**
```
【旁白】
"就在那雾气缭绕的山巅之上，一个古老的村落静静躺卧了千年。村口的老槐树见证了一代又一代人的悲欢离合，而今天，将有一场命运的邂逅在这里上演..."
```

## 注意事项

1. **分镜连续性**：下一个分镜的画面要自然衔接前一个分镜的结尾
2. **风格统一**：所有分镜使用一致的视觉风格（色调、光影）
3. **提示词质量**：英文提示词要详细、具体，包含风格标签
4. **旁白流畅**：旁白要像讲故事一样自然流畅
5. **分镜数量**：根据剧本长度，建议 8-16 个分镜，不宜过少或过多
